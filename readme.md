# 🔗 URL Shortener

A simple URL Shortener web application built using Python and Streamlit.

## 🚀 Features

- Shorten long URLs
- Generate short links
- Easy to use interface
- Fast URL conversion
- Simple and clean UI
- 🔐 User authentication
- 🗄️ Database integration
- 🎯 Custom short URLs
- 📊 URL click analytics

## 🛠️ Technologies Used

- Python
- Streamlit
- URL Shortening Logic

## 💻 Run Project On Your PC

Follow these steps to run the URL Shortener project on your local system.

---

### 1. Install Python

First, check Python is installed:

```bash
python --version
```

If Python is not installed, download it from:

https://www.python.org/downloads/

During installation, make sure to check:

```
☑ Add Python to PATH
```

---

### 2. Clone Repository

Open **Command Prompt / PowerShell** and run:

```bash
git clone https://github.com/chauhanmeet456-code/URL-Shortener.git
```

Go inside the project folder:

```bash
cd URL-Shortener
```

---

### 3. Create Virtual Environment (Recommended)

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

**Windows:**

```bash
venv\Scripts\activate
```

After activation you will see:

```
(venv)
```

---

### 4. Install Dependencies

Install required packages:

```bash
pip install -r requirements.txt
```

If `requirements.txt` is not available, install Streamlit manually:

```bash
pip install streamlit
```

---

### 5. Run Application

Start the Streamlit application:

```bash
streamlit run short_link.py
```

After successful execution, open:

```
http://localhost:8501
```

The URL Shortener application will open in your browser. 🚀

---

### 🛑 Stop Application

To stop the running application, press:

```
CTRL + C
```
